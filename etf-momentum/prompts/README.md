# ETF-Momentum 提示词工程（项目核心）

本目录包含策略的两大核心提示词，是"策略规则 → 大模型 → 交易决策"量化决策链的中枢。

## 两个提示词

| 文件 | 角色 | 频率 | 产出 |
|------|------|------|------|
| **`daily_prompt.txt`** | 交易员（CSO） | 每周一 | 调仓日报：摘要卡片 + 7模块分析 |
| **`review_prompt.txt`** | 策略官（CSO） | 每月 | 策略复盘：收益评估 + 信号诊断 + 优化建议 |

## 配套文件

| 文件 | 说明 |
|------|------|
| `daily_config.json` | 日报完整配置（system_prompt + 策略规则 + 7模块框架） |
| `daily_data_template.json` | 日报数据输入模板（6支ETF的量化数据） |
| `review_config.json` | 复盘完整配置（专家角色 + 6模块框架） |
| `review_data_template.json` | 复盘数据输入模板（收益、止损、过滤统计） |

## 角色约束（5原则 + 4禁止）

```
5原则：数据驱动 → 冷酷诚实 → 建设性批判 → 过拟合警觉 → 提示词工程视角
4禁止：模糊评价 | 外部信息 | 不可量化建议 | 无数字依据的结论
```

## 迭代闭环

```
daily_prompt.txt  ──→  大模型  ──→  调仓日报  ──→  执行交易
    ↑                                                │
    │                                    积累收益/止损数据
    │                                                │
    └─── 更新提示词  ←── 优化建议  ←── 复盘报告  ←── review_prompt.txt
```
